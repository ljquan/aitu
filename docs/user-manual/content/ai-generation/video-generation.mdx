# AI 生成视频

Opentu 支持多种 AI 视频生成模型，只需输入描述或提供图片，就能生成精彩的视频。

## 支持的模型

| 模型 | 特点 |
|------|------|
| **Veo3** | Google 视频模型，高质量生成 |
| **Veo3 Pro** | 增强版，更好的画质和细节 |
| **Veo3.1** | 支持首尾帧控制 |
| **Veo3.1 Pro** | 多图参考，更精确控制 |
| **Sora-2** | OpenAI 视频模型 |
| **Sora-2 Pro** | 增强版 Sora |

## 文字生成视频

最简单的方式，只需文字描述：

### 第 1 步：打开视频生成

1. 点击输入框旁边的 **#** 按钮
2. 切换到 **视频** 模式

### 第 2 步：输入描述

描述你想要的视频内容，如：

- "一只猫咪在草地上奔跑，阳光明媚"
- "城市街道的延时摄影，车流穿梭"
- "海浪拍打礁石，日落时分"

### 第 3 步：选择模型

点击模型选择器，选择合适的模型。

**建议**：
- 普通使用选 **Veo3** 或 **Sora-2**
- 需要高质量选 **Pro** 版本

### 第 4 步：发送生成

点击发送按钮，开始生成。

:::note
视频生成需要较长时间（通常 1-5 分钟），请耐心等待。生成过程中可以继续其他操作。
:::

## 图片生成视频

用一张图片作为起点，生成动态视频：

### 第 1 步：添加参考图

1. 点击输入框的 **图片** 图标
2. 选择本地图片或从素材库选择

### 第 2 步：描述动态效果

告诉 AI 你希望图片如何动起来：

- "让人物转头微笑"
- "让背景的云朵飘动"
- "镜头缓缓推进"

### 第 3 步：生成

选择支持图生视频的模型，发送生成。

## 首尾帧控制（Veo3.1）

需要更精确控制视频？使用首尾帧功能：

### 什么是首尾帧

- **首帧** - 视频开始的画面
- **尾帧** - 视频结束的画面

AI 会生成从首帧过渡到尾帧的视频。

### 如何使用

1. 选择 **Veo3.1** 或 **Veo3.1 Pro** 模型
2. 上传首帧图片（必须）
3. 上传尾帧图片（可选）
4. 描述过渡效果
5. 发送生成

**示例**：
- 首帧：日出前的天空
- 尾帧：太阳升起后的天空
- 描述："日出过程，光线逐渐变亮"

## 任务管理

### 查看进度

视频生成需要时间，可以在 **任务队列** 中查看进度：

1. 点击右下角的任务图标
2. 找到你的视频生成任务
3. 查看当前进度

### 后台生成

视频在后台生成，你可以：
- 继续其他创作
- 刷新页面也不会中断
- 关闭标签页后任务会继续

### 生成完成

生成完成后：
- 会收到通知
- 视频自动添加到素材库
- 可以插入到画布使用

## 提示词技巧

### 描述动作

- "缓慢移动" / "快速奔跑"
- "镜头推进" / "镜头拉远"
- "旋转" / "飘动" / "闪烁"

### 描述时长和节奏

- "3秒循环动画"
- "慢动作效果"
- "延时摄影风格"

### 描述氛围

- "电影质感"
- "梦幻效果"
- "复古滤镜"

## 常见问题

**Q：生成失败怎么办？**

A：
1. 检查网络连接
2. 检查 API 配置
3. 简化描述后重试
4. 在任务队列中点击"重试"

**Q：视频时长有限制吗？**

A：不同模型有不同限制，一般在 3-10 秒。

**Q：可以生成有声音的视频吗？**

A：部分模型支持，请参考具体模型说明。

**Q：视频可以下载吗？**

A：可以。在素材库中找到视频，右键保存或使用下载按钮。

## 小贴士

:::tip
视频生成比图片更耗时，建议先用图片生成确认效果，满意后再生成视频。
:::

:::tip
如果有特定的画面想要，先用 AI 生成图片，再用图片生成视频，效果更可控。
:::
