# 灵感创意板

不知道生成什么？灵感创意板提供各种创作模板，点击即可快速开始。

## 什么是灵感创意板？

当画布为空时，AI 输入框上方会自动显示灵感创意板，里面有各种预设的创作模板。

<Screenshot id="inspiration-step-1" alt="灵感创意板" />

每个模板都配有精心设计的提示词，只需点击就能直接使用。

## 如何使用

### 第 1 步：浏览模板

灵感创意板展示多种创作模板：

- **智能拆分宫格图** - 将图片拆分为九宫格
- **艺术风格** - 各种艺术风格的图片
- **实用场景** - 封面、海报、头像等

### 第 2 步：选择模板

点击你感兴趣的模板卡片。

<Screenshot id="inspiration-step-2" alt="选择模板后" />

模板的提示词会自动填充到输入框中。

### 第 3 步：调整发送

你可以：

- **直接发送** - 使用模板默认的提示词
- **修改提示词** - 根据需要调整描述
- **更换模型/尺寸** - 选择合适的设置

## 模板分类

### 智能拆分

适合制作：
- 朋友圈九宫格
- 拼图海报
- 系列图片组

**特点**：输入一段描述，AI 会自动生成可拆分的图片，一键拆成九宫格。

### 艺术创作

适合制作：
- 插画作品
- 概念艺术
- 创意海报

**特点**：预设了各种艺术风格的提示词，如水彩、油画、赛博朋克等。

### 实用场景

适合制作：
- 社交媒体配图
- 产品展示图
- 文章封面

**特点**：针对实际使用场景优化的模板。

## 小贴士

:::tip
灵感创意板会不定期更新新模板，记得常来看看有什么新创意！
:::

:::note
画布上有内容时，灵感创意板会自动隐藏。想再次查看？清空画布或新建项目即可。
:::

## 更多灵感来源

除了灵感创意板，你还可以：

1. **查看素材库** - 浏览之前生成的图片
2. **历史提示词** - 复用之前用过的描述
3. **参考其他作品** - 从网上找灵感，用自己的话描述
