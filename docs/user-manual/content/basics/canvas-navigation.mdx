# 画布操作

Opentu 提供无限画布，你可以自由拖动、缩放，拥有无限的创作空间。

<Screenshot id="canvas-tools" />
<p class="screenshot-caption">工具栏中的手形工具和选择工具</p>

## 移动画布

### 方法一：手形工具

1. 按 `H` 键切换到手形工具
2. 在画布上拖动

### 方法二：空格键 + 拖动

按住空格键，然后拖动鼠标。

### 方法三：鼠标中键

按住鼠标中键（滚轮按下）拖动。

## 缩放画布

### 方法一：鼠标滚轮

滚动鼠标滚轮即可缩放：
- 向上滚动 - 放大
- 向下滚动 - 缩小

### 方法二：快捷键

- `Ctrl/Cmd + +` 或 `Ctrl/Cmd + =` - 放大
- `Ctrl/Cmd + -` - 缩小
- `Ctrl/Cmd + 0` - 重置为 100%

### 方法三：视图导航

点击右下角的缩放控件：
- 点击 **+** 放大
- 点击 **-** 缩小
- 点击百分比数字，可以选择预设缩放比例

## 视图导航

<Screenshot id="view-navigation" />
<p class="screenshot-caption">视图导航控件：① 缩小 ② 自适应 ③ 放大</p>

右下角的视图导航提供了多种视图控制功能：

### 缩放控制

显示当前缩放比例，点击可选择：
- 50%、75%、100%、150%、200%
- **自适应** - 让所有内容适应屏幕

### 自适应视图

点击 **自适应** 或按 `Ctrl/Cmd + 0`，会自动调整缩放和位置，让所有内容都显示在屏幕内。

### 小地图

显示画布的整体缩略图，方便在大型画布中定位。

## 选择元素

### 选择单个元素

1. 按 `V` 键切换到选择工具
2. 点击要选择的元素

### 选择多个元素

**方法一**：框选

拖动鼠标画一个选择框，框内的元素都会被选中。

**方法二**：Ctrl/Cmd + 点击

按住 `Ctrl/Cmd` 键，逐个点击要选择的元素。

### 全选

按 `Ctrl/Cmd + A` 选择画布上所有元素。

### 取消选择

点击画布空白处，或按 `Esc` 键。

## 撤销和重做

做错了？没关系：

- **撤销** - `Ctrl/Cmd + Z`
- **重做** - `Ctrl/Cmd + Shift + Z`

:::tip
撤销/重做的历史记录会保存，可以多次撤销。
:::

## 复制和粘贴

### 复制元素

选中元素后：
- `Ctrl/Cmd + C` - 复制
- `Ctrl/Cmd + V` - 粘贴

### 复制到同一位置

- `Ctrl/Cmd + D` - 原地复制一份

### 拖拽复制

按住 `Alt` 键拖动元素，会复制一份新的。

## 删除元素

选中元素后：
- 按 `Delete` 键
- 或按 `Backspace` 键

## 快捷键汇总

| 快捷键 | 功能 |
|--------|------|
| `H` | 手形工具（移动画布） |
| `V` | 选择工具 |
| `空格 + 拖动` | 临时移动画布 |
| `Ctrl/Cmd + Z` | 撤销 |
| `Ctrl/Cmd + Shift + Z` | 重做 |
| `Ctrl/Cmd + A` | 全选 |
| `Ctrl/Cmd + C` | 复制 |
| `Ctrl/Cmd + V` | 粘贴 |
| `Ctrl/Cmd + D` | 原地复制 |
| `Delete` | 删除 |
| 滚轮 | 缩放 |
| `Ctrl/Cmd + 0` | 重置视图 |
