# 欢迎使用 Opentu（开图）

**Opentu（开图）** 是一款免费开源的 AI 图片视频创作工具 + 一体化白板。

**开**，代表开放、开源；**图**，象征视觉表达的无限可能。

无需安装，打开浏览器即可开始创作。

:::note
**重要提示：** 您的所有数据（项目、素材、提示词历史）都保存在**当前浏览器**的本地存储中，不会上传到服务器。这意味着：

- ✅ 数据隐私安全，完全由您掌控
- ⚠️ 换浏览器或清除浏览器数据会丢失内容
- 💡 需要迁移数据？使用 **菜单 → 备份恢复** 功能导出后在新浏览器导入
:::

## 我能用它做什么？

### AI 创作能力 🔥

- **AI 生成图片** - 支持多种图片生成模型
  - nano-banana - 快速生图
  - nano-banana-2 - 高质量生图，支持 HD/2K/4K 分辨率
  - 支持文本转图片、图片参考生成、批量生成

- **AI 生成视频** - 支持多种视频生成模型
  - Veo3 / Veo3 Pro - 高质量视频生成
  - Veo3.1 / Veo3.1 Pro - 支持首尾帧控制，多图参考
  - Sora-2 / Sora-2 Pro - OpenAI 视频模型
  - 支持图生视频、进度追踪、任务管理

- **模型自由切换** - 在生成对话框中即时切换不同的 AI 模型
- **任务队列管理** - 异步任务处理，支持批量生成、任务重试、历史记录、媒体缓存

### 白板与可视化

- **思维导图** - 快速整理思路，支持 Markdown 转换
- **流程图** - 绘制流程图，支持 Mermaid 语法
- **自由画笔** - 手绘涂鸦，多种画笔可选
- **形状工具** - 矩形、圆形、箭头等基础形状
- **图片插入** - 支持插入和编辑图片元素

### 编辑与交互

- **丰富的编辑功能** - 撤销、重做、复制、粘贴、多选等
- **无限画布** - 自由缩放、滚动、移动
- **自动保存** - 本地浏览器自动保存，数据不丢失
- **多格式导出** - 支持 PNG、JSON(`.drawnix`) 等格式导出

### 体验与生态

- **完全免费 + 开源** - MIT 许可证，可商用
- **主题模式** - 支持亮色/暗色主题切换
- **移动端适配** - 完美支持移动设备使用

## 立即开始

访问 [opentu.ai](https://opentu.ai) 开始使用，无需注册账号。

体验新功能？试试 [pr.opentu.ai](https://pr.opentu.ai)

## 快速入门

### 1. 生成图片

在底部输入框输入描述（如"一只可爱的猫咪"），点击发送。

### 2. 绘制思维导图

点击工具栏的思维导图图标，或按 `M` 键，开始创建。

### 3. 自由绘画

按 `P` 键切换到画笔，在画布上自由绘制。

## 数据存储说明

您的数据存储在浏览器本地，包括：

| 数据类型 | 存储位置 | 说明 |
|----------|----------|------|
| 项目文件 | IndexedDB | 画布内容和结构 |
| 素材库 | Cache Storage | 上传和生成的图片视频 |
| 提示词历史 | IndexedDB | 使用过的 AI 提示词 |
| 设置 | LocalStorage | API 配置等 |

**数据迁移方法：**

1. 在原浏览器打开 **菜单 → 备份恢复**
2. 勾选需要迁移的内容，点击 **导出**
3. 在新浏览器打开 Opentu，**导入** 刚才的 ZIP 文件

### 问题排查

遇到问题时，可以使用 **调试面板** 帮助定位：

| 环境 | 调试面板地址 |
|------|-------------|
| 正式环境 | [https://opentu.ai/sw-debug.html](https://opentu.ai/sw-debug.html) |
| 体验环境 | [https://pr.opentu.ai/sw-debug.html](https://pr.opentu.ai/sw-debug.html) |

调试面板可以：
- 查看 Service Worker 状态和日志
- 导出诊断日志（发给技术支持）
- 备份和恢复应用数据
- 清除缓存解决异常问题

:::tip
如需技术支持，请在调试面板中点击"导出日志"，将日志文件发送给开发者。
:::
