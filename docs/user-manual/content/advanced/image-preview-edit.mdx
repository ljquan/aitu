# 图片预览与编辑

Opentu 提供强大的图片预览和编辑功能，让你可以查看、裁剪、调整图片效果。

## 图片预览

<Screenshot id="image-preview" />
<p class="screenshot-caption">图片预览界面：① 关闭预览 ② 缩放控制 ③ 编辑工具</p>

### 打开预览

在画布上**双击**图片元素，即可打开全屏预览。

### 预览功能

预览界面支持：

- **缩放** - 滚轮缩放或点击 +/- 按钮
- **拖动** - 拖动查看大图细节
- **多图浏览** - 如果画布上有多张图片，可以左右切换
- **联动模式** - 多图时可开启同步缩放/拖动（按 `S` 键切换）

### 预览工具栏

| 按钮 | 功能 |
|------|------|
| 缩放滑块 | 调整图片大小 |
| 适应窗口 | 让图片适应当前窗口 |
| 原始大小 | 以 100% 比例显示 |
| 上一张/下一张 | 切换图片 |
| 联动模式 | 多图同步操作 |

## 图片编辑

### 进入编辑模式

1. 在画布上选中图片
2. 点击弹出工具栏中的 **编辑** 按钮（画笔图标）

或者在预览界面中点击 **编辑** 按钮。

### 编辑界面

图片编辑器提供两大功能：

## 裁剪功能

点击 **裁剪** 按钮进入裁剪模式：

### 自由裁剪

直接拖动裁剪框的边缘或角落，调整裁剪区域。

### 比例裁剪

选择预设的宽高比：
- **自由** - 任意比例
- **1:1** - 正方形
- **4:3** - 传统照片比例
- **16:9** - 宽屏比例
- **9:16** - 竖屏比例
- **原始** - 保持原图比例

### 智能去白边

点击 **自动去白边** 按钮，自动检测并裁剪掉图片的白色边框。

:::tip
这个功能对 AI 生成的图片特别有用，可以快速去除多余的白边。
:::

## 滤镜调整

点击 **滤镜** 按钮进入调整模式：

### 预设滤镜

选择预设滤镜效果：
- **原图** - 恢复原始效果
- **鲜艳** - 增强饱和度和对比度
- **柔和** - 降低对比度，更柔美
- **复古** - 怀旧暖色调
- **黑白** - 经典黑白效果
- **冷色** - 蓝色冷调
- **暖色** - 橙色暖调

### 手动调节

使用滑块精细调整：
- **亮度** - 整体明暗
- **对比度** - 明暗差异
- **饱和度** - 色彩鲜艳程度
- **色温** - 冷暖色调
- **锐化** - 图片清晰度

## 保存编辑

编辑完成后，有三种保存方式：

### 覆盖原图

点击 **保存** 按钮，直接覆盖画布上的原图。

### 另存为新图

点击 **另存为** 按钮，保留原图，在画布上创建新的图片。

### 下载

点击 **下载** 按钮，将编辑后的图片保存到本地。

### 放弃编辑

点击 **X** 关闭按钮或按 `Esc` 键，放弃所有修改。

## 撤销和重做

编辑过程中支持撤销重做：

- **撤销** - 点击工具栏的撤销按钮，或按 `Ctrl/Cmd + Z`
- **重做** - 点击重做按钮，或按 `Ctrl/Cmd + Y`

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + Z` | 撤销 |
| `Ctrl/Cmd + Y` | 重做 |
| `Esc` | 取消/退出编辑 |
| `Enter` | 确认操作 |

## 小贴士

:::tip
编辑大图时可能需要等待一会儿，请耐心等待处理完成。
:::

:::tip
如果对编辑效果不满意，可以随时撤销或重新开始。
:::

:::note
SVG 格式的图片暂不支持编辑功能。
:::
