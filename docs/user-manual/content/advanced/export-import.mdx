# 导出与导入

Opentu 支持多种格式的导出和导入，方便你保存和分享作品。

## 导出图片

### 导出为 PNG

1. 点击左上角的菜单按钮
2. 选择 **导出图片**
3. 选择 **PNG 格式**

**特点**：
- 支持透明背景
- 适合网页使用
- 文件较大

**快捷键**：`Ctrl/Cmd + Shift + E`

### 导出为 JPG

1. 点击菜单按钮
2. 选择 **导出图片**
3. 选择 **JPG 格式**

**特点**：
- 不支持透明背景
- 文件较小
- 适合照片类图片

## 导出选中区域

只想导出一部分？

1. 先选中要导出的元素
2. 再执行导出操作

只有选中的内容会被导出。

## 保存项目

### 保存为 JSON

将项目保存为 `.drawnix` 文件（实际是 JSON 格式）：

1. 点击菜单按钮
2. 选择 **保存文件**
3. 选择保存位置

**快捷键**：`Ctrl/Cmd + S`

### 打开项目

1. 点击菜单按钮
2. 选择 **打开文件**
3. 选择 `.drawnix` 文件

## 自动保存

Opentu 会自动保存你的工作：

- 数据保存在浏览器本地
- 刷新页面不会丢失
- 切换项目自动保存

:::tip
虽然有自动保存，但重要项目建议定期手动导出备份。
:::

## 备份与恢复

### 导出备份

可以将所有数据打包导出：

1. 点击菜单按钮
2. 选择 **备份 / 恢复**

<Screenshot id="backup-restore" alt="备份恢复对话框" />

3. 选择要备份的内容：
   - 提示词历史
   - 项目文件
   - 素材库
4. 点击 **导出**

会下载一个 ZIP 文件，包含所有选中的数据。

### 导入备份

从备份文件恢复：

1. 点击菜单按钮
2. 选择 **备份恢复**
3. 选择要导入的 ZIP 文件
4. 选择要恢复的内容
5. 点击 **导入**

:::note
导入会与现有数据合并，不会覆盖。
:::

## 分享作品

想分享你的作品？

1. **导出为图片** - 分享到社交媒体
2. **导出为 JSON** - 发给其他 Opentu 用户
3. **截图分享** - 快速分享

## 常见问题

**Q：导出的图片太大/太小？**

A：导出的图片大小取决于画布上内容的实际尺寸。如果需要特定尺寸，可以先调整内容大小。

**Q：导出时背景是透明的？**

A：PNG 格式支持透明背景。如果需要白色背景，可以在画布底层添加一个白色矩形，或导出为 JPG 格式。

**Q：项目文件可以发给别人吗？**

A：可以。把 `.drawnix` 文件发给对方，对方用 Opentu 打开即可。
