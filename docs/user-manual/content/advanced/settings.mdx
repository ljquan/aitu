# 设置与配置

了解如何配置 Opentu 的各项设置，包括 API 配置、主题切换等。

## 打开设置

点击左上角的菜单按钮，打开应用菜单：

<Screenshot id="app-menu" alt="应用菜单" />

选择 **设置** 打开设置对话框：

<Screenshot id="settings-dialog" alt="设置对话框" />

## API 配置

要使用 AI 生成功能，需要配置 API。

### 图片生成 API

1. 打开设置 → **图片生成** 选项卡
2. 填写 **API Key**
3. 填写 **API 地址**（如有需要）
4. 点击保存

**获取 API Key**：
- 可以使用 [tu-zi.com](https://tu-zi.com) 提供的 API 服务
- 或使用自己的 Gemini API Key

### 视频生成 API

1. 打开设置 → **视频生成** 选项卡
2. 配置视频 API 相关信息
3. 保存设置

## 主题切换

支持亮色和暗色两种主题：

1. 点击右下角的主题切换按钮
2. 或在设置中选择主题模式

**亮色主题**：适合白天使用，背景明亮
**暗色主题**：适合夜间使用，保护眼睛

## 语言设置

支持中文和英文界面：

1. 打开设置
2. 找到语言选项
3. 选择 **中文** 或 **English**

## 数据存储

### 本地存储

Opentu 的数据存储在浏览器本地：

- **项目文件** - 你的画布内容
- **素材库** - 上传和生成的图片视频
- **提示词历史** - 使用过的提示词
- **设置** - 你的配置信息

### 存储位置

数据保存在：
- IndexedDB - 大文件存储
- LocalStorage - 配置信息
- Cache Storage - 缓存数据

### 数据安全

- 数据不会上传到服务器
- 只保存在你的浏览器中
- 清除浏览器数据会丢失

:::tip
重要数据请定期使用 **备份恢复** 功能导出备份。
:::

## 清除数据

如需清除数据：

### 清除当前画布

1. 点击菜单 → **清空画布**
2. 确认清除

**快捷键**：`Ctrl/Cmd + Backspace`

### 清除所有数据

1. 打开浏览器设置
2. 清除网站数据
3. 选择 opentu.ai 的数据

:::note
清除后无法恢复，请先备份重要内容。
:::

## 快捷键设置

目前使用系统默认快捷键，暂不支持自定义。

常用快捷键参考首页的快捷键列表。

## 移动端适配

Opentu 支持在移动设备上使用：

- **触摸操作** - 支持手指缩放和拖动
- **响应式布局** - 自动适应屏幕大小
- **工具栏适配** - 底部工具栏优化

### 移动端特殊操作

- **双指缩放** - 缩放画布
- **单指拖动** - 移动画布（手形模式）
- **长按** - 触发右键菜单

## 常见问题

**Q：设置保存在哪里？**

A：保存在浏览器的 LocalStorage 中，换浏览器需要重新配置。

**Q：API Key 安全吗？**

A：API Key 只保存在你的浏览器本地，不会上传到服务器。但请注意不要在公共电脑上配置。

**Q：为什么 AI 功能不可用？**

A：请检查：
1. API Key 是否正确配置
2. API 地址是否正确
3. 网络是否正常
4. API 额度是否充足

**Q：如何迁移到新电脑？**

A：使用 **备份恢复** 功能：
1. 在旧电脑导出备份 ZIP 文件
2. 在新电脑导入该 ZIP 文件
