# 素材库

素材库用于管理你的图片和视频素材，包括 AI 生成的和手动上传的。

## 打开素材库

点击底部工具栏的 **素材库** 按钮，或点击底部的图库图标。

<Screenshot id="media-library" alt="素材库面板" />

## 素材来源

素材库中的素材有两种来源：

### AI 生成

所有通过 AI 生成的图片和视频都会自动保存到素材库。

### 手动上传

你也可以手动上传图片：

1. 打开素材库
2. 点击 **上传** 按钮
3. 选择本地图片文件

## 使用素材

### 插入到画布

1. 在素材库中找到想要的图片
2. 点击图片
3. 图片会自动插入到画布中

### 作为参考图

在 AI 生成图片时：

1. 点击输入框的图片图标
2. 选择 **素材库**
3. 选择要参考的图片

## 管理素材

### 查看详情

点击素材可以查看大图和详细信息。

### 删除素材

1. 选中素材
2. 点击删除按钮

:::note
删除素材不会影响已经插入画布的图片。
:::

### 筛选素材

可以按类型筛选：
- **全部** - 显示所有素材
- **图片** - 只显示图片
- **视频** - 只显示视频

## 存储说明

素材保存在浏览器本地存储中：

- **优点**：无需登录，数据在本地，隐私安全
- **注意**：清除浏览器数据会丢失素材

:::tip
重要素材建议使用 **备份恢复** 功能定期导出备份。
:::

## 常见问题

**Q：素材库满了怎么办？**

A：可以删除不需要的素材释放空间。或者导出备份后清理。

**Q：换电脑/换浏览器素材还在吗？**

A：不在。素材保存在当前浏览器中。需要使用 **备份恢复** 功能迁移数据。

**Q：素材库支持多大的文件？**

A：取决于浏览器的存储限制，一般几百 MB 到几 GB 不等。
