🚀 部署步骤

  1. 推送代码到GitHub：
  git add netlify.toml
  git commit -m "add netlify deployment config"
  git push origin main
  2. 在Netlify中配置：
    - 登录 https://netlify.com
    - 点击 "New site from Git"
    - 连接你的GitHub仓库
    - 选择分支（通常是main或develop）
    - Netlify会自动读取netlify.toml配置
  3. 自动部署触发：
    - 每次推送到指定分支时自动构建和部署
    - Pull Request会创建预览部署
    - 支持分支部署预览
