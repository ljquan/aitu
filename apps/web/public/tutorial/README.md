# Tutorial Assets

新手引导教程资源文件夹。

## 文件结构

```
/tutorial/
├── version.txt      # 版本号文件（热更新关键）
├── ai-image.mp4     # AI 生图教程视频
└── ai-video.mp4     # AI 生视频教程视频
```

## 热更新说明

视频文件替换后**立即生效**，无需重新部署。

**更新步骤：**
1. 上传新视频文件（保持相同文件名）
2. 修改 `version.txt` 中的数字（如 `1` → `2`）
3. 用户刷新页面即可看到新视频

**原理：**
- 代码会读取 `version.txt` 获取版本号
- 视频 URL 自动附加 `?v=版本号`
- 版本号变化 = URL 变化 = 重新加载

## 缓存策略

| 文件 | 缓存时间 | 说明 |
|-----|---------|------|
| `version.txt` | 不缓存 | 每次检查最新版本 |
| `*.mp4` | 永久缓存 | URL 变化时自动更新 |

## 视频规格建议

- **分辨率**: 720p (1280x720) 或更低
- **格式**: MP4 (H.264 + AAC)
- **时长**: 5-15 秒
- **文件大小**: < 5MB
